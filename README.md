# YYXTest

Demo link: https://chenyuhsu413.github.io/YYXTest/

## Project Overview

This is a static website for 楊宜修國文文理補習班. The page introduces the class programs, after-school tutoring services, classroom photos, contact information, and social links.

## Features

- Responsive single-page layout
- Top announcement marquee
- Sticky navigation bar
- Classroom photo gallery
- Contact cards for phone, Line, and Facebook
- Google Maps link for the school address
- Back-to-top button
